We're going to spend most of our time reviewing loops and conditionals today. However, in the last bit of class we'll talk about one last  topic in **Control Flow**: the idea of event listeners. These "listeners" are just special functions that we write for Python to run when it _hears_ certain events. This could be something like a mouse click or someone typing on the keyboard...or you can think of something more advanced like a thermometer dropping below a certain value or a toner cartridge becoming empty.

Though today we'll focus our time on MOUSE events, next week we'll talk about KEYBOARD events.
