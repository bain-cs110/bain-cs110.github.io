Just like the value of a <em>variable</em> might affect the outcome of a <em>command</em>, a <strong>parameter</strong> is a value that can be used as input to a program to change the way it behaves. Below we have our usual turtle sim, but with an important added feature, a slider at the top called NUMBER-OF-LAVA.

Change the slider and then click the RESET button. Now do it again. See any differences?
